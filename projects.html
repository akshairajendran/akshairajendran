sportbot

sportbot is an automated market making bot for Sportcrypt Ethereum Sports Betting Smart Contracts

It prices and maintains two-sided markets in all Sportcrypt contracts using a live odds feed from JSONOdds and uses historical game data from SportsDatabase.com to price over/under and point spread bets

For more information please see my github
